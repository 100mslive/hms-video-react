# VideoTile

You can use VideoTile component to render local or remote video content.

## Importing

```javascript
import { VideoTile } from '100ms-sdk-component-library-react';
```

## Example

### Default VideoTile

<Canvas>
  <DefaultVideoTile
    videoSource="camera"
    peer={{ id: '123', displayName: 'Eswar' }}
    {...DefaultVideoTile.args}
  />
</Canvas>

Note: You need to pass MediaStream object as props to VideoTile for it to work

```jsx
<VideoTile
  isLocal={true}
  stream={MediaStream}
  peer={{ id: '123', displayName: 'Eswar' }}
  aspectRatio={{ width: 1, height: 1 }}
  displayShape="rectangle"
  showAudioLevel={true}
  audioLevelDisplayType="border"
  audioLevel={40}
/>
```
